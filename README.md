# GPG Notebook

This is a demo application using Ruby on Rails 5.0 to build a GPG/PGP encrypted notebook for safe note storage.

| Rails 5         |              |             | Vagrant       | Backups    |
| Nginx           |              |             |               |            |
| Passenger       |              |             |               |            |
| Mongoid         |              |             |               |            |
